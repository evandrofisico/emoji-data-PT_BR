# emoji-data

Fork do excelente https://github.com/Mange/emoji-data, que gera dados 
de emojis programaticamente, mas em Português do Brasil


Para mais dados veja o [README-orig.md]


## Implementaćão

Assim como o original, essa aplicaćão baixa os dados oficiais do unicode,
que incluem nomes e descrićões dos emojis em portugês, porém as categorias
e subcategorias são extraídas de um arquivo de teste, todos em inglês.

Então eu apenas remendei a aplicaćão para usar apenas os arquivos
de português e traduzi o arquivo de dados de teste, de onde originalmente
são puxadas as categorias.

## Uso

**tl;dr:** `make all`

### Dependencies

Para gerar os arquivos, precisa instalar:

1. Ruby (3.1 ou superior).
2. `bundler` (gerenciador de dependências GEM de Ruby)
3. `jq` Processador de json de linha de comando

Rodando `make setup` as dependências de ruby devem ser instaladas

### Gerando dados

Rode `make` para gerar todos os dados. Ainda estou testando, então só posso
garantir  que os seguinte targets funcionem
`make data/all.json`
`make data/all_emojis.txt`
